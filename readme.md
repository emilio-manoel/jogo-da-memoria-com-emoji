# Desafio auxiliado dio - desenvolvimento um jogo de memoria com emoji
Esse desafio foi com auxilio de video aulas, que na primeira versão promove uma brincadeira simples de memoria; no qual se utiliza: <br> 
<br>
**HTML:** Em uma abordagem mais resumida e simples; containers, titulos, botões e classes.
<br>
**CSS:** uma estrutura mais avancaçada com ultilização de um reset para se adaptar a todos os navegadores, conceitos como ajuste de tamanho de imagem, de possisionamento, mudanças de cores, fontes etc.
<br>
**JavaScript:**  também mais desenvolvido, com a criação de um  armazenamento para os emoji, além de embaralhar os emoji, verificar se são iguais e se todos os emojis foram encontrados dar uma mensagem de parabéns, juntamente com a opção de reiniciar o jogo.
<br>
